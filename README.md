# Juego de Adivinanza de Números en Python 🎯

Un juego de adivinanza donde debes descubrir un número aleatorio entre 1 y 100.

## ¿Cómo funciona?
1. Seleccionas la dificultad
2. El programa genera un número aleatorio entre 1 y 100
3. Tienes un número limitado de intentos para adivinarlo
4. El programa te dice si el número es mayor o menor al que ingresaste
5. Si adivinas ganas, si no, se revela el número correcto

## Dificultades
| Dificultad | Intentos |
|------------|----------|
| Fácil | 10 |
| Medio | 5 |
| Difícil | 3 |

## Requisitos
- Python 3.x

## Cómo ejecutarlo
```bash
python adivinanza.py
```

## Ejemplo de uso
```
¡Vamos a jugar! Estoy pensando en un número entre 1 y 100.
Selecciona la dificultad (fácil, medio, difícil): fácil
Ingresa tu número (10 oportunidades restantes): 50
El número es mayor que el que ingresaste.
Ingresa tu número (9 oportunidades restantes): 75
¡Felicidades! 75 es el número correcto.
```
